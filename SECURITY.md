# Security Policy

## Supported Versions

Only the latest major version receives security fixes.

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Reporting a Vulnerability

If you discover any security related issues, please use [GitHub Security Advisories](https://github.com/tempi-marathon/blade-meteocons/security/advisories/new) instead of the public issue tracker.
